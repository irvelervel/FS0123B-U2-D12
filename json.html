Appunti sul formato JSON

JSON sta per JavaScript Object Notation. È un formato di interscambio per applicazioni client/server.

Lo useremo ogni volta che avremo la necessità di recuperare dati da internet o se avremo bisogno di inviare dei dati a nostra volta al fine di venire memorizzati in un database passando per un server.

JSON non è stato "inventato"! Nasce dalla notazione letterale di JS

Regole da tenere a mente quando si scrive un file JSON:

  1. I nomi delle chiavi devono sempre essere delimitate da doppi apici
  2. Tutti i valori saranno unicamente proprietà e non metodi
  3. Tutte le coppie chiave/valore saranno separate da virgola, a parte l'ultima di un oggetto o di un array
